# Contributions Guide for Pero's Academy GitHub Pages Welcome to Pero's Academy! This guide will help you set up your own GitHub Pages repository to create and host your personal website. ## Getting Started 1. **Create Your Repository**: Go to the [Pero's Academy Repositories](https://github.com/orgs/Pero-s-Academy/repositories) and create a new repository named `your_username.github.io`. Replace `your_username` with your actual GitHub username. 2. **Clone the Repository**: Clone the repository to your local machine using Git. ``` git clone https://github.com/Pero-s-Academy/your_username.github.io.git ``` ## Building Your Website ### Index Page - **Create `index.html`**: This is the main page of your website. Use HTML to structure your content and CSS to style it. - **Content**: Introduce yourself, showcase your projects, and highlight your skills. ### Additional Pages - You can create additional HTML pages such as `about.html`, `contact.html`, and so on. - Ensure that you link these pages correctly from your `index.html`. ## Tips for Your Site - **Keep It Simple**: Start with a simple layout and enhance it as you go. - **Mobile Friendly**: Ensure your site is responsive and looks good on all devices. - **SEO**: Use relevant keywords and descriptions to improve your site's visibility. ## Hosting Your Site - **Commit Your Changes**: Use Git to add, commit, and push your changes to GitHub. - **Go Live**: Once pushed, your site will be live at `https://your_username.github.io`. ## Goals of This Exercise 1. **Promote Yourself**: Use this platform to highlight your talents and professional journey. 2. **Learn and Share**: Gain valuable knowledge about web development and version control. 3. **Expand Your Skills**: Utilize these skills to contribute to larger projects like `github.pero.co.za`. ## Examples for Inspiration - [sui.github.io](https://github.com/Pero-s-Academy/sui.github.io) - [elliott29.io](https://github.com/Pero-s-Academy/elliott29.io) - [burnt-exe.github.io](https://github.com/burnt-exe/burnt-exe.github.io) Take a look at [Raydo Matthee's Kickresume](https://www.kickresume.com/cv/raydo-matthee/) for ideas on what you can create. ## Conclusion Use this project to present yourself in the best light, learn how to host websites on GitHub for free, and apply your new skills to create subdomains like `github.pero.co.za`. Let's get started!
